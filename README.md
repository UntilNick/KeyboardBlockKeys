# KeyboardBlockKeys

Программа для блокировка клавиш на клавиатуре! 

Список клавиш для блокировки можно дополнять в `List<Keys> downKeys = new List<Keys>`

P.S: Не блокирует сочитание клавиш.

Протестировано на *Windows 10 x64*
